## Project Overview
Server Bot is a backend chatbot application built using **Flask**, **MongoDB**, and **Redis**.  
It handles user interactions, manages conversation history, and integrates with a **Generative AI API** to generate intelligent responses.

The project is designed as a server-side chatbot system with session handling, user validation, and persistent storage for conversations and user data.

---

## Tech Stack
- **Backend Framework:** Flask (Python)
- **Database:** MongoDB
- **Cache / Session Store:** Redis
- **Frontend:** HTML (Flask templates)
- **AI Integration:** Google Gemini API
- **Security:** Password hashing using Werkzeug
- **Others:** UUID, JSON, Datetime, Validators
